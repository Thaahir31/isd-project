<task type="auto">
  <name>Task 2: Approval Logic with SELECT FOR UPDATE</name>
  <files>src/api/loans.py</files>
  <action>
    Implement `POST /api/loans/requests/{request_id}/approve`:
    1. Start a transaction.
    2. Fetch the `LoanRequest` by ID.
    3. Fetch the `Book` record using `query(Book).filter(...).with_for_update()`.
    4. Validate:
       - Request is PENDING.
       - Book `available_copies > 0`.
    5. Action:
       - Set `LoanRequest.status = APPROVED`.
       - Decrement `Book.available_copies`.
       - Create `Loan` record (status=ACTIVE).
    6. Commit transaction.
  </action>
  <verify>
    <automated>pytest tests/test_lending_approval.py (Verify successful approval and state changes)</automated>
  </verify>
  <done>Librarians can approve requests, resulting in active loans and updated inventory.</done>
</task>
