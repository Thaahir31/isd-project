<task type="auto">
  <name>Task 2: Return Book Endpoint</name>
  <files>src/api/loans.py</files>
  <action>
    Implement `POST /api/loans/{loan_id}/return`:
    - Requires `LIBRARIAN` role.
    - Inputs: `condition` (Good, Damaged, Lost).
    - Logic:
      1. Fetch `Loan` by ID (ensure status is ACTIVE).
      2. Calculate fine using `src/logic/fines.py`.
      3. Update `Loan`:
         - status = RETURNED.
         - returned_at = now.
         - condition = input.condition.
         - total_fine = calculated_fine.
      4. Fetch `Book` and increment `available_copies`.
      5. Commit.
  </action>
  <verify>
    <automated>pytest tests/test_returns.py (Verify return process and inventory restoration)</automated>
  </verify>
  <done>Librarians can process returns, restore inventory, and see calculated fines.</done>
</task>
